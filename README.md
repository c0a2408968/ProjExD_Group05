# こうかとん、講義に遅刻する

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1
* 必要なものがあれば追記してください（非推奨）

## ゲームの概要
* 主人公キャラクターこうかとんが大学の講義に遅刻しそうなために爆走するゲーム
* 段差や穴をスペースキーでジャンプして回避！

## ゲームの遊び方
* 段差や穴をスペースキーでジャンプして回避
* バスはたまに上から降ってくるのでダブル、トリプルジャンプで回避しよう
* 出てくる先生キャラを拾った武器で倒すとゴールしなくてもクリアになる（打つときはエンターキー）
* 落ちている卵を集めるとスピードが上がる
* 段差やバスにぶつかったり穴に落ちたりするとゲームオーバー

## ゲームの実装
### 共通基本機能
* 背景画像（落とし穴や段差）と主人公キャラクター（ジャンプ、落ちる、ぶつかる）の描画

### 分担追加機能
* ゴール、ゲームオーバー描写（担当：橋本）：ゴールやゲームオーバーの状況によって描写を変える
* 卵（担当：菊地）：卵を拾うとカウントされ、７個卵を拾うと主人公のスピードがあがる
* バス（担当：古跡）：たまに空から降ってきて主人公を邪魔し、バスの横か下にぶつかるとゲームオーバーとなるが、バスの上にぶつかってもセーフ判定となる
* 先生（担当：三池）：低確率で出てくる先生で、武器で倒すと休校になりゴールしなくてもクリアになる
* 武器（担当：柴田）：たまに武器が落ちており、先生やバスを攻撃できる
